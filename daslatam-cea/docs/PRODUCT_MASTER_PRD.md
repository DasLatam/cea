# DAS LATAM CEA

## Comercio Electrónico en Argentina

**Tipo:** Plataforma web de inteligencia para e-commerce  
**Modelo:** Gratuita + monetización por anuncios  
**Infraestructura base:** Next.js + Vercel + Supabase

## Propósito

Detectar oportunidades reales de negocio en Argentina combinando:

- demanda
- competencia
- proveedores
- costos
- fiscalidad
- logística
- timing
